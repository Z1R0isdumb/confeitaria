# Site - Oficina de Confeitaria Infantil | Cake Break Orlando

Site estático feito com HTML, CSS e JavaScript puro, inspirado no cartaz enviado.

## O que está incluído

- Landing page responsiva
- Aesthetic divertida/pastel com confeitos, etiquetas e cards arredondados
- Cupcake interativo com confetti
- Mini game de coletar confeitos
- Cards viráveis das 4 aulas
- Seleção de pacote/aula
- Formulário em modal que monta mensagem automática no WhatsApp
- Countdown para Early Bird até 20 de junho de 2026
- Seção com o cartaz original como referência visual

## Como editar rápido

No arquivo `script.js`, edite o objeto `CONFIG` para mudar:

```js
const CONFIG = {
  whatsappNumber: "13213189311",
  instagram: "cakebreakorlando",
  eventYear: 2026,
  earlyBirdDeadline: "2026-06-20T23:59:59-04:00"
};
```

No `index.html`, edite textos, valores e receitas.

## Como abrir

Abra o arquivo `index.html` no navegador.

## Como publicar

Você pode publicar os arquivos em Netlify, Vercel, GitHub Pages ou qualquer hospedagem estática.
